TCP/IP
La pareja de protocolos básicos de Internet. Permiten conectar de manera redundante y segura a los ordenadores que se encuentran conectados a la red mundial. Fue creado a principio de los años 70 en EEUU con objetivos militares.
HTTP
HyperText transfer Protocol o Protocolo de Transferencia de Hipertexto. Es el protocolo basado en texto que utilizan los servidores y los navegadores Web para comunicarse y transmitir entre sí los contenidos de las páginas, imagenes y otros archivos que se intercambian en la Web.
DNS
Domain Name Server
Domain Name Server, es el protocolo que se encagra de traducir nombres de dominio de páginas Web en direcciones IP de los servidores que las sirven. Así, por ejemplo, www.google.com se corresponde con la dirección IPv6 "2a00:1450:4002:802::1013", que identifica a los servidores del buscador de manera única en Internet.
